## File: `state.red`

```Red
Red []

parser: context [
	state: []
	rules: [
		any [
			number! (append state 'number) |
			word!   (append state 'word)   |
			time!   (append state 'time)
		]
	]
	run: func [input] [
		state: copy []
		parse input rules
		state
	]
]

probe parser/run [1 2 3]
probe parser/run [x 5 y 10:30]
```
