## File: `count-lines.red`

```Red
Red[]

count-lines: func [
	filename
	/local
		file count ws
][
	ws: charset " ^-"
	file: read/lines filename
	count: 0
	foreach line file [
		unless any [
			empty? line
			parse line [some ws]
			parse line [any ws #";" to end]
		][
			count: count + 1
		]
	]
	count
]

```
