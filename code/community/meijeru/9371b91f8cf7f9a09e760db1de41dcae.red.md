## File: `to-local-time.red`

```Red
to-local-time: func [
  "adjusts a time given as UTC, or a date containing a time, to the local timezone"
  d [time! date!]
][
  either time? d
  [
    d + now/zone
  ][
    if d/timezone [
      d/timezone: now/zone
    ]
    d
  ]
]
```
