## File: `flatten.red`

```Red
flatten: func [tree [block!] /level lvl /local rule l][
    (l: -1)
    rule: [(l: l + 1) some [
       ahead block! if (any [not level l < lvl]) into rule (l: l - 1) 
    |  keep skip
    |  none
    ]] 
    parse tree [collect rule]
]

```
